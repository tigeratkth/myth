# 碳陶瓷刹车盘技术资料

> 来源：Google Patents（CN102661342A、CN115823151B、WO2017107735A1）、汽车制动网（2022.06）、Molando Brake 技术博客。本段为公开资料摘录整理。

## 1. 技术原理与材料组成

碳陶瓷刹车盘（C/SiC 或 CSiC）是一种**碳纤维增强陶瓷基复合材料**（Carbon-Fiber Reinforced Ceramic Matrix Composite），核心成分三件套：

- **碳纤维**：提供强度与韧度，防止基体脆性开裂；
- **碳化硅（SiC）**：构成陶瓷基体，贡献硬度与高温稳定性；
- **硅粉 / 石墨**：作为渗透相与固体润滑，决定最终摩擦系数。

工作时，刹车片与盘面之间形成**摩擦转移层**，热量被碳纤维网络沿径向快速疏散，避免"热衰退"现象。

## 2. 典型制造工艺（深圳勒马 WO2017107735A1 所述路线）

总耗时约 3 周，关键工序：

1. 碳纤维原料与**酚醛树脂**混合后放入模具；
2. 约 **200°C 高温 + 20,000 kg 压力** 压制成型；
3. **1,000°C** 裂解加热两天（将树脂碳化成碳基体）；
4. 添加硅粉后在**真空条件**下第三次加热，温度达 **1,700°C**、持续 24 小时，熔融硅与碳反应生成 SiC；
5. 形成增强型碳化硅复合陶瓷毛坯，再机加工成最终盘面。

中南大学 CN102661342A 还公开了"炭纤维增强陶瓷基汽车刹车片"的替代路线；西安超马 CN115823151B 主打"夹心结构的炭/陶刹车盘"，通过分层设计在同一盘体实现**外层高硬度 + 内层高韧性**。

## 3. 典型性能参数

| 指标 | 数值 |
| --- | --- |
| 摩擦系数 μ | 0.44 – 0.52 |
| 稳定工作温度 | 750 – 800°C（部分产品可承受近 900°C） |
| 重量 | 约为同规格铸铁盘的 55%，旋转质量下降 ~45% |
| 冷却速度 | 通风设计下较铸铁盘提升 21% |
| 使用寿命 | 25 万 – 30 万 km（接近整车寿命） |
| 0–100 km/h 制动距离 | 相比铸铁盘缩短约 3 m |

## 4. 相比铸铁 / 钢盘的差异化

- **热稳定性高** — 连续弯道制动不掉性能；
- **簧下质量低** — 对操控与乘坐舒适性均有直接增益；
- **低粉尘、耐腐蚀** — 轮毂外观长期洁净；
- **寿命对齐整车** — 全生命周期几乎免更换。

劣势：单盘成本显著高于普通盘，对刹车片材料有强制配套要求；低温冷态摩擦系数较低，需要首圈"烘盘"。

## 5. 应用场景

主要搭载于高性能跑车（保时捷 PCCB、法拉利 CCB）与高端电动旗舰（蔚来 ET7 选装、Lucid Air、部分保时捷 Taycan 版本）。近年随国产工艺成熟，开始下探到 50 万元级性能车型。
